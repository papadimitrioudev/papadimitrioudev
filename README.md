# Papadimitriou.dev

A personal portfolio website showcasing my projects and professional experience.

## About

Portfolio site for Panagiotis Papadimitriou - Computer Science Graduate and Data Analytics Student.

## Features

- **Responsive Design** - Optimized for desktop, tablet, and mobile devices
- **Dynamic Starfield Background** - Animated space-themed background
- **Project Showcase** - Detailed project descriptions with live links
- **Professional CV** - Harvard-style resume with print-to-PDF functionality

## Technologies

- HTML5
- CSS3 (Custom animations, responsive design)
- Vanilla JavaScript
- No frameworks or dependencies

## Projects Featured

- **Galactic Run** - 3D endless runner game (Unity, C#)
- **Storm Haven** - Weather application with bilingual support
- **Reading Marathon** - Real-time leaderboard system with WebSocket

## Local Development

1. Clone the repository
2. Open `index.html` in your browser
3. No build process required
